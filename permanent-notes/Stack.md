# Stack

A stack is a linear data structure that supports adding elements to one end and removing elements from that same end.

| Operation | Big-O |
|-----------|-------|
| Top/Peek | O(1) |
| Push | O(1) |
| Pop | O(1) |
| Search | O(n) |
| isEmpty | O(1) |
